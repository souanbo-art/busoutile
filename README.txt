このパッケージは、1スクロールLPの完成形（見本）です。

1) 置換する文字
index.html 内の次の3つを置き換えてください。
__LINE_URL__ ＝ LINE公式アカウントのURL
__TEL__      ＝ 電話番号（ハイフン可）
__ADDRESS__  ＝ 住所（市区町村まででも可）

2) 画像について
assets の画像は「雰囲気の見本」です。
実運用では、権利を確認した写真、または自社施工写真に差し替えてください。
差し替えは assets/ の画像ファイルを入れ替えるだけで構いません。

3) GitHub Pages での公開
このフォルダの中身をそのままリポジトリ直下へ置き、GitHub Pages を有効化してください。
